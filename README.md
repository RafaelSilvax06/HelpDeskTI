# 🎫 HelpDeskTI

Sistema de Help Desk desenvolvido em **C#** com **arquitetura em camadas** para gerenciamento de chamados de suporte técnico.

---

# 📋 Sobre o Projeto

O **HelpDeskTI** foi criado para centralizar e organizar solicitações de suporte dentro de uma empresa.

O sistema permite que usuários abram chamados e acompanhem o andamento do atendimento, enquanto analistas conseguem visualizar e gerenciar todas as solicitações.

---

# 🎯 Objetivos do Projeto

* Aplicar conceitos de **Programação Orientada a Objetos (POO)**
* Implementar **separação de responsabilidades em camadas**

# 🚀 Funcionalidades

## 👤 Cliente

* Abertura de chamados
* Acompanhamento do status de atendimento
* Abertura e exclusão de chamados

## 🛠️ Analista

* Visualização de chamados
* Gerenciamento e processamento de solicitações
* Cadastro de novos clientes ou analistas

# 🚀 Como Executar

## 1. Clonar o Projeto

```bash
git clone https://github.com/RafaelSilvax06/HelpDeskTI.git
```

## 2. Executar Localmente

Certifique-se de ter o **SDK do .NET** instalado em sua máquina.

Abra o projeto no **Visual Studio** ou **VS Code**.

No terminal, dentro da pasta do projeto, execute:

```bash
dotnet run
```
